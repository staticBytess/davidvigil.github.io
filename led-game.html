MY PROJECTS

Reaction Time Game

My first project working with a Raspberry Pi and Python. Three LED bulbs are placed in a row with a physical button in front of each one. The Pi lights a random bulb and times how long the user takes to press the correct button.

Outcomes:
  • Learned to program hardware with Python.
  • Learned to wire the components correctly and ensure a short would not occur.
Challenges:
  • Wiring the components.
  • Finding libraries to work between the code and hardware.
Process:
The first step was learning Python, then writing the components correctly, and finally programming the components with Python.
