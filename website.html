This Website

This website you are using right now is my most interactive project yet.
A technical challenge as well as a writing and art design challenge all packed into one project.

Objectives:
  • Have a one stop shop for all my projects.
  • Learn HTML, CSS, and Javascript to make a fun, interactive website.
  • Learn how to host a website.
Outcomes:
  • Experience with Github.
  • General experience working front end.
Challenges:
  • Learning three new languages at once.
  • Wanting to implement many features all at once.
Process:
Started with a basic HTML page with text and an image and grew from there. I included JS to program features like the top navigation menu and *future* dropdowns. It has been a lot of fun experimenting with the front end.
