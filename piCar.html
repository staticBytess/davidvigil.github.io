Raspberry Pi Pico "Car"

A wifi-controlled “car” powered by a Raspberry Pi Pico.
Initial scope: Assemble and program an RC car to be controlled over a LAN connection.

Outcomes:

  • Learned how to use Python to speak with motor drivers.
  • Created a LAN connection with Python.
  • Increased confidence working on embedded systems.

Challenges and Solutions:

Challenge #1: Struggled to program the motor drivers.
Solution: Used libraries to help with the process.

Challenge #2: Power usage was too low for the battery pack to register and would shut off.
Solution: Connected an extra display I had to ensure enough power was being pulled.

Process:

Purchased the components off of Amazon, used a cardboard sheet as the frame, and a Raspberry Pi Pico to power it all. The components consisted of: wheels, motors and drivers, batteries, and the Pico components. Started with programming a single motor then scaled it up to handle two pairs. Various versions were made during development, from a single wheel to a pair of wheels on a half-sized cardboard sheet, ultimately to the final build.

Additional Notes:

  • Steering and wheel alignment were not in the scope of this project.
